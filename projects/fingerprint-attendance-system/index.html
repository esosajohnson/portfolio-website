    • Fingerprint-Based Attendance System

    A robust attendance management system utilizing fingerprint recognition technology to ensure accurate and efficient attendance tracking.

    Project Overview

    This Fingerprint-Based Attendance System is designed to streamline the process of tracking attendance through biometric fingerprint recognition. The system captures fingerprint data, verifies identities, and records attendance in a secure and efficient manner. It features a user-friendly interface, which allows administrators for easy management of attendance records, reporting, and analytics. The system is built with a focus on security, accuracy, and ease of use, making it an ideal solution for organisations looking to enhance their attendance management processes.

    Project Approach

    The system was developed through a structured process that began with selecting suitable biometric hardware and communication methods. The device software was written in C/C++ using Arduino IDE, integrating Wi-Fi connectivity, fingerprint processing, and OLED feedback. A local server was built using XAMPP, with PHP and MySQL managing data storage, user accounts, and attendance logs. JavaScript supported the web dashboard’s interactivity. The system was tested across core scenarios—including enrollment, verification, and reconnection—to ensure dependable operation.

    Project Features

    • Biometric Fingerprint Recognition
    • Secure Communication with Server
    • User Authentication and Authorisation
    • Real-time Attendance Tracking
    • Comprehensive Reporting and Analytics
    • Easy-to-Use User Interface

    Skills Used

    • Programming: C/C++, PHP, SQL, JavaScript
    • Embedded Systems: Arduino UNO R4 WiFi, OLED (I2C), R307 sensor (UART)
    • Database Management: MySQL (XAMPP)
    • Network Communication: Wi-Fi, HTTP GET and POST communication
    • Software Tools: Arduino IDE, phpMyAdmin
    • Project Management: Time planning (Gantt chart), documentation, presentation delivery

    Benefits

    • Accurate Attendance Tracking: Reduces errors associated with manual entry.
    • Enhanced Security: Biometric verification ensures only authorized personnel can record attendance.
    • Efficiency: Streamlines the attendance process, saving time for both employees and administrators.
    • Wireless Connectivity: Enables real-time data transfer between hardware and server.
