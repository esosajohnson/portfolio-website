    • Sales Categorisation Using Machine Learning

    A machine learning project that classifies sales data into distinct categories using Support Vector Machines, Neural Networks, and Decision Trees.

    Project Overview

    This project focuses on categorising sales data into different segments using various machine learning algorithms, including Support Vector Machines (SVM), Neural Networks, and Decision Trees. The objective is to analyse historical sales data, identify patterns, and classify the data into meaningful categories that can help businesses make informed decisions. The project involves data preprocessing, feature selection, model training, and evaluation to ensure accurate classification results. By leveraging these machine learning techniques, the project aims to provide insights into sales trends and customer behaviour, ultimately aiding in strategic planning and marketing efforts.

    Project Approach

    The development of the Sales Categorisation Using Machine Learning project began with a thorough understanding of the problem and its requirements. The project involved data preprocessing, feature selection, model training, and evaluation to ensure accurate classification results. Various machine learning algorithms, including Support Vector Machines (SVM), Neural Networks, and Decision Trees, were used to classify sales data into distinct categories. The project utilised libraries such as NumPy, Pandas, and Scikit-Learn to handle data manipulation and machine learning tasks. The project concluded with a technical presentation outlining the design choices, results, and overall functionality.

    What Was Delivered

    This project delivered a production-ready pipeline for product category classification, including cleaned and preprocessed sales data, engineered features, and three supervised models (SVM with polynomial kernel, a Neural Network/MLP, and a Decision Tree) tuned via Grid/Randomized Search. It also includes clear evaluation (accuracy, F1, confusion matrices), executive-friendly visualisations of trends and model performance, and concise documentation explaining the approach, assumptions, and results, plus a final presentation summarising key insights and recommended next steps.
